# AI Annotation Portfolio

## Overview
This repository showcases representative AI annotation and NLP workflow examples focused on data quality, sentiment analysis, intent classification, and dataset validation.

## Focus Areas
- AI Data Annotation
- Natural Language Processing (NLP)
- Sentiment Analysis
- Intent Classification
- Dataset Validation
- Quality Assurance
- Data Labeling

## Purpose
The goal of this portfolio is to demonstrate structured annotation workflows, quality review practices, and language data evaluation methods used in AI and machine learning projects.

## Projects Included
- Sentiment Analysis Workflow
- Intent Classification Workflow
- Dataset Quality Review
- Text Annotation Examples
